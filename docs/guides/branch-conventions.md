## 快速参考

### 分支命名

```
feature/<description>   新功能
fix/<description>        问题修复
refactor/<description>   重构
docs/<description>       文档更新
chore/<description>      维护类改动
```

### 提交格式

```
<type>: <简要描述>
```

| type | 含义 |
|------|------|
| `feat` | 新功能 |
| `fix` | 问题修复 |
| `refactor` | 重构 |
| `docs` | 文档更新 |
| `test` | 测试相关 |
| `chore` | 维护、依赖、配置 |
