# 项目路线图

> 记录项目各阶段的完成状态与后续计划。

---

## 系统定位演进

项目从**轻量 WMS**出发，当前进入向**轻 ERP + WMS 一体化**演进的阶段。

- **WMS 核心闭环**（已完成）：商品/客户/供应商主数据 + 手工入库/出库单 + 库存管理 + AI 辅助识别
- **ERP 方向**（演进中）：单据工作台体验（录单效率）→ 采购单/销售单 → 财务摘要

当前成熟度：核心业务闭环可运行，前端单据工作台重构完成（composables 层 + 键盘录入流），后端 Service 层覆盖率 91.73%（80% 门槛已落地 CI），工程化基础设施与 GitHub 仓库保护链已就绪。
